# Piolho OptimizeX Site

Site oficial da Piolho OptimizeX com tema dark/verde premium.

## Rodar em desenvolvimento

```bash
npm install
npm run dev
```

Abra o link mostrado no terminal, normalmente:

```bash
http://localhost:8080
```

## Gerar build

```bash
npm run build
```

## Testar build

```bash
npm run preview
```

## Páginas principais

- `/` — Home
- `/dashboard` — Download
- `/discord` — Discord
- `/privacy-policy` — Política de Privacidade
- `/terms-of-service` — Termos de Serviço
